# Swaggify

Add OpenAPI 3.0 definitions to your Grip application and document your API.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     swaggify:
       github: grip-framework/swaggify
   ```

2. Run `shards install`
